Class DecryptedItem

Wraps Items returned from the API that have been decrypted, usually by ItemService. If associations, classification_nodes, thumbnails and attachmentsare provided at construction, they are stored.

DecryptedItem is immutable, you should use toItemUpdate to stage modifications.

Note that when using fromAPI to construct a DecryptedItem classification_nodes, associations_to and associations are filtered to just those that concern the Item. thumbnails and attachments are NOT filtered; they may contain extra attachments not used by the Item.
